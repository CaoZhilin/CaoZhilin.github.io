    Thonghts about Web Font-End Technology Class
    Web 前端课程感想

    在大一的时候曾经上过计算机系的网页设计的课程，但是这门通选课的内容非常的简单和基础，更多只是涉及了静态页面的html、css的内容和一些flash的制作。并且课程教学使用了dreamweaver这种现在已经较少使用的软件。在前端的内容中我们感兴趣的除了好看的样式之外更多的是JavaScript所实现的交互和动态效果。

    在以前的两个学年里，我们所涉及的有关图形交互的编程只有MFC和Qt，而网页前端的制作是更有吸引力的。昨天课上内容讲的JavaScript如果是结合了网页交互的例子的话我觉得可能会更容易理解，并且因为有带入感从而更有吸引力，但是老师所偏重的恰恰是语言特性。

    对于作业来说，也觉得很有趣，首先以做作业的方式写一个个人主页非常有纪念意义，以及大作业实现一个前端的小游戏也是一件神奇事。和大力姐姐组队之后，我们点开了童年的回忆-4399和摩尔庄园来寻找合适类型的小游戏（炫耀摩尔庄园的装备），最后竟然看起了英雄联盟的直播，非常有意思的经历。经过一个晚上的讨论，我们已经有了初步的方案，所以也非常期待实现之后的效果。

    未来对于课程的期待希望可以有一些好的前端的资源和教程以及比较新的技术能够由老师和助教分享到网络学堂上，供我们课后学习。
